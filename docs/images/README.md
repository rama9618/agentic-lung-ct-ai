# Screenshot Placeholders

Place polished demo screenshots for the GitHub README in this folder.

Expected files:

```text
docs/images/dashboard.png
docs/images/gradcam.png
docs/images/pseudo_3d.png
docs/images/chatbot.png
```

Recommended capture checklist:

- `dashboard.png`: full Streamlit dashboard with upload, prediction, and sidebar visible
- `gradcam.png`: original CT image, Grad-CAM heatmap, and overlay
- `pseudo_3d.png`: 3D tumor attention map with dark background
- `chatbot.png`: AI Medical Assistant answering a useful question

Keep screenshots free of private patient information.
